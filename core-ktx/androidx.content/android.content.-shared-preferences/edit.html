edit

inline fun SharedPreferences.edit(commit: Boolean = false, action: Editor.() -> Unit): Unit

Allows editing of this preference instance with a call to apply or commit to persist the changes. Default behaviour is apply.

prefs.edit {
    putString("key", value)
}

To commit changes:

prefs.edit(commit = true) {
    putString("key", value)
}
